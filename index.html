About the App

Hi, I'm James Tang and thank you for playing! Originally started as a simple card-matching game, I decided to add a Peanuts theme because I've been a fan of Snoopy since childhood. I used HTML, CSS & Bootstrap to display and style the game while using JavaScript (written in OOP) for game functionality. As one of the first projects started when I began to learn how to program, I struggled with applying JavaScript and jQuery for the card logic and DOM creation. I came back to start the project from scratch after 2 months and was happy to find that what challenged me initially had turned into a piece of cake. Please feel free to visit my GitHub or Portfolio for more of my projects. Thank you!
